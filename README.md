# Bitburner Scripts

A collection of scripts for Bitburner; focused on automation, efficiency, and learning the mid- to late-game mechanics. 

These scripts are designed to: 
- Automate and optimize early-game sections
- Serve as examples you can modify and extend

This is a toolbox. 

## Repository Structure 
```
├── hacking/          # Hack / grow / weaken loops and orchestrators
├── contracts/        # Coding contract solvers
├── hacknet/          # Hacknet automation & optimization
├── ipvgo/            # IPvGO helpers and managers
├── utils/            # Shared helpers (network scans, formatting, math)
├── startup/          # Scripts meant to run on reset / install
└── README.md
```
