# Bitburner Scripts

A collection of scripts for Bitburner; focused on automation, efficiency, and learning the mid- to late-game mechanics. 

These scripts are designed to: 
- Automate and optimize early-game sections
- Serve as examples you can modify and extend

Please note this is a toolbox. 

## Repository Structure 
```
├── hacking/          # Hack / grow / weaken loops and orchestrators
├── contracts/        # Coding contract solvers
├── hacknet/          # Hacknet automation & optimization
├── ipvgo/            # IPvGO helpers and managers
├── utils/            # Shared helpers (network scans, formatting, math)
├── startup/          # Scripts meant to run on reset / install
└── README.md
```
## Recommended Run Order (New Saves Only)

1. **Manual Play (First Minutes)**
    - Gain initial RAM
    - Buy TOR
    - Open basic ports
2. **Startup Scripts**
    - Network scan
    - Root access scripts
    - Baseline weaken loops
3. **Basic Hacking Scripts**
    - Single-target automation
    - Money generation stabilizations
4. **Hacknet Automation**
    - Use with spending limits
    - Monitor ROI
5. **Contracts**
    - Solve opportunistically
    - Use solvers as helpers
6. **IPvGO (Once Unlocked)**
    - Run helpers, not full automation at first
7. **Scaling**
    - Purchased servers
    - Orchestrators
    - Optimization passes
