# Coding Contract Solvers

Automated and semi-automated solvers for Bitburner Coding Contracts. 

Covers common contracts types: 
- Algorithmic Stock Trader
- Generate IP Address
- Total Ways to Sum
- Spiral Matrix
- LZ Decompression
- Hamming Codes

## Use Notes

Some solvers:
- Only print answers
- Require manual copy / paste
- Auto-submit (labeled)

Not every solver handles every contract. Always test on a low-stakes contract first.
