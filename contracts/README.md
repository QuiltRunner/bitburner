# Coding Contract Solvers

Automated and semi-automated solvers for Bitburner Coding Contracts. 

Covers common contracts types: 
- Algorithmic Stock Trader
- Generate IP Address
- Total Ways to Sum
- Spiral Matrix
- LZ Decompression
- Hamming Codes

## File Structure
```
├── solve-contracts.js
└── solvers/
    ├── index.js
    ├── stock-trader.js
    ├── ip-addresses.js
    ├── total-ways.js
    ├── spiralize.js
    ├── triangle.js
    ├── unique-paths.js
    └── prime-factor.js
```
## Use Notes

Some solvers:
- Only print answers
- Require manual copy / paste
- Auto-submit (labeled)

Not every solver handles every contract. Always test on a low-stakes contract first.
